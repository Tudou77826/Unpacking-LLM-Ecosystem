# 🚀 解构 LLM：从第一性原理到驾驭工程 (Unpacking LLM Ecosystem)

> **“当 AI 变成一种通用基础设施，我们的任务不再是学习如何成为它，而是学习如何驾驭它。”**

## 📖 关于本仓库

欢迎来到本仓库！建立这里的初衷，是为了拒绝盲目的概念追逐，回归 AI 技术的本质。

我们将采用 **“第一性原理”** 和 **“苏格拉底式追问”** 的学习方法，彻底搞清楚大语言模型（LLM）生态的演进脉络。在这里，我们不只是罗列代码和工具，而是探究每一个技术节点的底层逻辑：**它是什么？它解决了上一个阶段的什么痛点？它又是怎么演变出下一个新技术的？**

通过本仓库，你将和我一起走过一条极为清晰的技术进化之路：从理解模型的“大脑引擎”，到教会它“听话”，再到给它“喂数据”、“装手眼”，最后走向 Autonomous Agent（自治体）时代的终极考验——**建构护栏（Harness Engineering）**。

## 🗺️ 内容导航 (The Evolution Roadmap)

本仓库的学习路线严格按照技术演进的逻辑划分为**五个核心阶段**：

### 🧱 阶段一：大模型的基石 (LLM 原理) —— *造引擎*
*痛点追问：机器究竟是如何“思考”和“说话”的？*
* **本源解析**：把 Transformer 拆碎，通俗理解 Next Token Prediction（文字接龙）的数学逻辑。
* **涌现与对齐**：从 Pre-training（预训练）到 SFT（微调）再到 RLHF（人类反馈强化学习），模型是如何获得逻辑能力并符合人类价值观的？
* **能力边界**：彻底搞懂为什么 LLM 必然存在“幻觉”（Hallucination）。

### 🗣️ 阶段二：对话的艺术 (Prompt Engineering) —— *握方向盘*
*痛点追问：既然模型有了智能，我们该如何准确下达指令（turn right）？*
* **基础心法**：零样本（Zero-shot）、少样本（Few-shot）的底层机制。
* **思维激发**：为什么加一句“让我们一步步思考”就有奇效？深入剖析 CoT (Chain of Thought) 思维链的运作原理。
* **局限性**：提示词工程的尽头是什么？为什么光靠“话术”无法解决专业知识匮乏的问题？

### 📚 阶段三：记忆与投喂 (Context Engineering) —— *给地图*
*痛点追问：如果模型不知道昨天的内部数据，我们怎么把信息安全地“喂”给它？*
* **RAG的诞生**：检索增强生成（Retrieval-Augmented Generation）的工作流解析。
* **上下文困境**：为什么把几百万字全塞进上下文（Context Window），模型会变“笨”（Context Rot/Lost in the Middle）？
* **工程解法**：如何系统化地策划、过滤、排序模型在推理时看到的信息（向量数据库、分块策略）？

### 🛠️ 阶段四：赋予手眼与统一协议 (Skill & MCP) —— *装轮子与通用接口*
*痛点追问：光说不练假把式，如何让 LLM 去真实世界里“办事”（查天气、写文件、调API）？*
* **化身为 Agent**：Skill/Tool Calling 是如何实现的？模型是如何判断“何时该调用哪个函数”的？
* **思维到行动**：ReAct（推理+行动）框架的底层推演。
* **破局者 MCP (Model Context Protocol)**：
  * 为什么过去的插件生态一地鸡毛？
  * 深入解析 MCP：AI 时代的“通用 USB 接口”。我们如何通过 MCP 无缝对接本地文件、数据库和企业内部系统？

### 🛡️ 阶段五：走向自治时代的护栏 (Harness Engineering) —— *修公路与建护栏*
*痛点追问：当 Agent 可以自主写出 100 万行代码时，人类该如何控制那些我们甚至看不懂的复杂系统？*
* **角色转换**：为什么工程师的工作从“写代码”变成了“设计环境”？
* **驾驭工程三大支柱**：
  * **架构约束 (Architectural Constraints)**：如何用结构化测试和 linter 给 Agent 立规矩，防止它乱来？
  * **反馈闭环 (Feedback Loops)**：如何让模型自己审查模型，形成自动纠错机制？
  * **熵管理 (Entropy Management)**：对抗 AI 生成产物的快速腐化（Garbage Collection）。
* **终极思考**：在未来，Human-in-the-loop（人在回路）的具体形态是什么？

## 🎯 适合人群

* **架构师与决策者**：想要看清从 Prompt 到 Harness 的技术代差，从而构建企业级 AI 系统的思考者。
